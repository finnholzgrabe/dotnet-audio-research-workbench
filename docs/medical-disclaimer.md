# Medical Disclaimer

This project is for software-engineering, signal-processing, and machine-learning learning purposes only.

It is not:

- a medical device
- a cochlear implant simulator
- a clinical decision-support tool
- a validated auditory model
- a hearing-aid or cochlear implant fitting tool

Any cochlear-inspired feature extraction in this repository must be described as an approximation for experimentation, not as a model suitable for medical or clinical use.
