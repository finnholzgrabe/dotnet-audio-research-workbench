Cochlear Audio Research Workbench

A small .NET-first research-software workbench for audio feature extraction, cochlear-inspired filter-bank features, and a reproducible toy ML baseline.

C# / .NET 8 DSP FFT / STFT MFCC cochlear ERB bank k-NN baseline 61 tests

Source on GitHub · Not a medical device, cochlear-implant simulator, or validated auditory model.

Pipeline & results

What it does

  • Deterministic WAV fixtures & 16-bit PCM IO
  • Framing, Hann window, radix-2 FFT, STFT
  • Cochlear-inspired (ERB) filter bank + MFCCs
  • k-NN baseline with stratified / k-fold / leave-speakers-out evaluation
  • Optional real data: Free Spoken Digit Dataset (CC BY-SA 4.0)

What it is not

  • Not a medical or clinical tool
  • Not a validated auditory model
  • Not production-grade ML

The cochlear filter bank is a documented engineering approximation.

Quick start

dotnet build AudioResearch.sln -c Release
DLL=src/AudioResearch.Cli/bin/Release/net8.0/AudioResearch.Cli.dll

dotnet "$DLL" generate tone --freq 440 --seconds 1 --out samples/generated/tone.wav
dotnet "$DLL" inspect samples/generated/tone.wav
dotnet "$DLL" ml baseline --out artifacts/baseline-report.json
