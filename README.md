# Cochlear Audio Research Workbench

Planned publishable project scaffold.

This repository is intended to become a .NET-first research-software workbench for experimenting with audio feature extraction, cochlear-inspired filter-bank features, and a reproducible toy ML baseline.

It is not a medical device project, not a cochlear implant simulator, and not a clinically validated model.

## How To Use This Scaffold

Open this folder in Claude Code or another coding agent and follow [CLAUDE.md](CLAUDE.md).

The build should happen step by step:

1. Initialize repository hygiene and documentation.
2. Add the .NET solution skeleton.
3. Add deterministic audio fixtures and WAV IO.
4. Add the DSP feature pipeline.
5. Add the ML baseline.
6. Polish for publication.

## Intended Evidence

When complete, this project should honestly demonstrate:

- C#/.NET research-software structure
- tested audio/DSP primitives
- reproducible generated audio fixtures
- cochlear-inspired feature extraction
- a small ML baseline
- clear docs and limitations

Do not claim these until they exist in code and tests.
